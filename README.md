# Tu Apple Online - Tienda en Línea de Apple FRONTEND

## Descripción
"Tu Apple Online" es una tienda en línea que ofrece una amplia selección de productos Apple, como iPhones, MacBooks, iPads, Apple Watch y más. Este proyecto utiliza tecnologías modernas como React, Vite, React Bootstrap, JavaScript y FontAwesome para crear una experiencia de compra en línea atractiva y receptiva.


## Características
- **Catálogo de Productos**: Presenta un catálogo completo de productos Apple con imágenes, descripciones detalladas y precios.

- **Carrito de Compras**: Los usuarios pueden explorar productos, agregarlos al carrito y realizar compras seguras.

- **Diseño Receptivo**: El sitio web se adapta perfectamente a una variedad de dispositivos, lo que garantiza una experiencia de usuario óptima.

- **React y Vite**: Utiliza React como biblioteca de interfaz de usuario y Vite como un entorno de desarrollo ultrarrápido.

- **React Bootstrap**: Aprovecha React Bootstrap para agilizar el desarrollo y asegurar un aspecto moderno y agradable.

- **FontAwesome**: Incorpora íconos FontAwesome para mejorar la interfaz de usuario con íconos elegantes.

## Instrucciones de Uso
1. Clona este repositorio en tu máquina local utilizando Git:
   ```
   git clone https://github.com/valentinalbornozz/Tu-Apple-Online-en-react.git
   ```
2. Ve al directorio del proyecto:
   ```
   cd Tu-Apple-Online-en-react
   ```
3. Instala las dependencias utilizando npm o yarn:
   ```
   npm install o yarn
   ```
4. Inicia la aplicación en modo de desarrollo:
   ```
   npm run dev o yarn dev
   ```
5. Abre tu navegador y visita http://localhost:5173 para ver el sitio web en acción.
6. Explora el catálogo de productos, agrega productos al carrito y realiza compras como usuario.

7. Tecnologías Utilizadas
React
Vite
React Bootstrap
CSS
